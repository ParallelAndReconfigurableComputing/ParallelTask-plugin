Preface

This file describes how to install the PTJava Editor Plug-In into the Eclipse IDE.

Requirements

  • Eclipse SDK Version 3.5.1 or Greater

Installation Instructions

Please ensure that any previous installations of the PTJava Text Editor Plug-in have been removed prior to installing.

Installation Using the PTJava Feature

  1. Download PTJavaFeature.zip
  2. Open Eclipse
  3. Navigate to Help >> Install New Software...
  4. Click Add...
  5. Select Archive... and navigate to the PTJavaFeature.zip file. This will add the archive to the list of sites to check for features
  6. Make sure the "Group items by category" option is unchecked
  7. The PTJava Feature should now appear, and can be selected for installation.
